1. ¿Qué hace st.session_state?
	Es como una memoria temporal de la app,
	Sirve para guardar datos entre clics y acciones del usuario.

2. ¿Qué hace calcular_subtotal()?
	Cada vez que compras un producto, lo añade a la lista de compras.  

3. ¿Por qué se usa st.form()?
	El formulario hace que la compra se registre solo cuando presionas “Comprar Producto”.
   
4. ¿Qué se muestra con st.dataframe()?
	El carrito de compras en pantalla, actualizado en tiempo real.
